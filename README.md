**Officially Supported OS's**
- Ubuntu 20.04 LTS (Recommended)
- Ubuntu 18.04 LTS
- Ubuntu 16.04 LTS

**For Ubuntu 20.04 LTS:**
```
git clone https://github.com/kis2a/UNIT3D-INSTALLER.git installer
cd installer
sudo ./install.sh
```
